HTML概念介绍

1. 概念：

超文本标记语言 Hyper Text Markup Language(HTML)

HTML 是最基础的网页开发语言

超文本：

超文本是用超链接的方法，将各种不同空间的文字信息组织在一起的网状文本

标记语言：

  • 由标签构成的语言。 <标签名称> 如 html, xml
  • 标记语言不是编程语言
